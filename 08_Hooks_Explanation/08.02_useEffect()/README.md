## Summary

The `useEffect` hook allows React components to perform **side effects** — actions that happen outside normal UI rendering, such as fetching data, updating titles, or running timers.
It replaces the old lifecycle methods like `componentDidMount`, `componentDidUpdate`, and `componentWillUnmount` from class components.

This document explains each concept clearly with **4 practical examples** to help you understand when and how to use `useEffect`.
