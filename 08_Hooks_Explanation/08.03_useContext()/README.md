## Summary

The `useContext` hook allows React components to **share data globally** without passing props manually through every level.
It works together with `createContext()` and `Provider` to create a shared data environment that can be accessed by any component in the tree.

This document clearly explains how to:

- Create a context (`createContext`)
- Provide data (`Provider`)
- Consume data (`useContext`)

It also demonstrates a practical **Theme Toggle App** using the `useContext` hook to switch between light and dark modes.
