## Summary

The `useReducer` hook is used to manage **complex state logic** in React components.
It works similarly to `useState`, but provides a **more structured and predictable** way to handle state updates — especially when multiple actions affect the same state.

This document clearly explains how to:

- Define a reducer function
- Use `dispatch()` to trigger actions
- Manage state transitions based on `action.type`

It also includes two practical examples:

1. **Counter App** — Increase, decrease, and reset values using `useReducer`.
2. **User Profile App** — Manage object-based state (name & age) using reducer logic.
