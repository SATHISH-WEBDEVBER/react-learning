---

## React Hooks Overview

React Hooks are **special functions** that let you use React features (like state, lifecycle, and context) inside **functional components** — without writing class components.

React Hooks என்பது **சிறப்பு செயல்பாடுகள்** (functions) ஆகும், இது React இல் உள்ள **state, lifecycle, context** போன்ற அம்சங்களை **functional components**-இல் பயன்படுத்த அனுமதிக்கிறது, class components எழுதாமல்.

---

## 16 Hooks List

### Basic Hooks

1. **useState** — Manage state (data) inside functional components**useState** — Functional components-இல் தரவை (state) நிர்வகிக்க உதவும்
2. **useEffect** — Perform side effects (fetch data, timers, etc.)**useEffect** — வெளிப்புற செயல்களை (API call, timer போன்றவை) இயக்க பயன்படும்
3. **useContext** — Share data easily across components**useContext** — கூறுகளுக்கு இடையே தரவை எளிதாகப் பகிர்ந்து கொள்ள உதவும்
4. **useRef** — Access DOM elements or persist mutable values**useRef** — DOM கூறுகளை அணுக அல்லது மதிப்புகளை நிலையாக வைத்திருக்க பயன்படும்
5. **useMemo** — Optimize performance by caching calculations**useMemo** — கணக்கீடுகளை சேமித்து செயல்திறனை அதிகரிக்க உதவும்
6. **useCallback** — Prevent unnecessary re-renders for functions
   **useCallback** — தேவையற்ற மீள்-ரெண்டரை (re-render) தடுக்க உதவும்

---

### Additional Hooks

7. **useReducer** — Alternative to useState for complex state logic**useReducer** — சிக்கலான state logic க்கான useState க்கு மாற்றாகும்
8. **useLayoutEffect** — Runs before browser paints (for layout updates)**useLayoutEffect** — Browser layout-ஐப் புதுப்பிப்பதற்கு முன் இயக்கப்படும்
9. **useImperativeHandle** — Customize values exposed by refs**useImperativeHandle** — Ref மூலம் வெளியிடப்படும் மதிப்புகளை கட்டுப்படுத்த பயன்படும்
10. **useDebugValue** — Helps to display debug info for custom hooks**useDebugValue** — Custom hooks க்கான debug தகவலை காட்ட உதவும்
11. **useId** — Generate unique IDs for elements**useId** — கூறுகளுக்கு தனிப்பட்ட ID உருவாக்க உதவும்
12. **useTransition** — Handle state transitions smoothly**useTransition** — State மாற்றங்களை மென்மையாக நடத்த உதவும்
13. **useDeferredValue** — Delay updates to improve performance**useDeferredValue** — செயல்திறனை மேம்படுத்த update களை தாமதப்படுத்தும்
14. **useSyncExternalStore** — Sync React with external data sources**useSyncExternalStore** — React ஐ வெளிப்புற தரவுடன் ஒத்திசைக்க உதவும்
15. **useInsertionEffect** — Inject styles before rendering**useInsertionEffect** — Rendering க்கு முன் CSS styles சேர்க்க பயன்படும்
16. **Custom Hooks** — Create your own reusable hooks
    **Custom Hooks** — உங்கள் சொந்த ஹூக்ஸ் உருவாக்கி மீண்டும் பயன்படுத்தலாம்
